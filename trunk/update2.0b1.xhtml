PirateQuesting

Version: 2.1b1

This is a beta! It will have bugs and it will quite possibly annoy you. It is also, however, the only option as the last version is crashing/freezing Firefox.

Features of this release include:

  • Drag and Drop Tabs
  • New code for almost every part of the extension
  • Not just a sidebar! The extension continues to work while the sidebar is closed.
  • Updated fight and training logging. Although they look similar now, the data being collected is much more extensive than the old versions and have more room for growth in ways to display/report the data.
  • and much more.
